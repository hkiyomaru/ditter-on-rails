###Versions
- Ruby 2.1.7
- Rails 4.0.2

###Getting started

Get the code.

```
$ git clone https://github.com/kiyomaro927/Ditter_on_Rails.git
```

Install the gem packages.

```
$ bundle install
```

Execute migration.

```
$ bundle exec rake db:migrate
```

###Run

Run ditter on WEBrick.

```
$ rails server
```
